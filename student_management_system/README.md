# Student Management System
> My first complete Python project built as part of my software engineering learning journey.

A simple command-line Student Management System built with Python.

This project was created to practice Python fundamentals, file handling, JSON storage, functions, input validation, and basic software development principles.

## Features

- Add a student
- View all students
- Search a student by ID
- Update student details
- Delete a student
- Find the student with the highest CGPA
- Sort students by CGPA
- Store data permanently using JSON

## Technologies Used

- Python 3
- JSON
- Git
- GitHub

## How to Run

Clone the repository:

```bash
git clone <repository-url>
```

Go to the project folder:

```bash
cd student_management_system
```

Run the program:

```bash
python student_management_system.py
```

or

```bash
py student_management_system.py
```
